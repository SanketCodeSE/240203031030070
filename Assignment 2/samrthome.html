1.Smart Home(Automatic System)

Steps:-

Step-1 : Collect components: Arduino, relay module, ESP8266, and jumper wires.

Step-2 : Connect the ESP8266 for Wi-Fi communication.

Step-3 : Wire the relay module to control appliances.

Step-4 : Program the Arduino using the Arduino IDE.

Step-5 : Configure the ESP8266 to connect to home Wi-Fi.

Step-6 : Create a smartphone app or web interface for control.

Step-7 : Test by turning devices on/off via the interface.

Step-8 : Install securely and ensure proper insulation.
